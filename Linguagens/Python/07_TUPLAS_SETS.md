gi## • **Tuplas**

° Tuplas assim como as **listas, dicionários e variáveis** são estruturas para dados voláteis.

° Uma das características únicas das **tuplas** é a não aceitação de alteração sobre os dados que nelas forem inseridos.

° Normalmente utilizadas para leituras de repostas do **python** mas com a possibilidade de uso para um sistema de **logs** por exemplo.

° São representadas pelo o uso de: **()**
